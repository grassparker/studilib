## 🚀 StudiLib: The Social Productivity Engine

**StudiLib** is a gamified, real-time productivity platform designed to turn solitary study sessions into a collaborative experience. It bridges the gap between traditional "To-Do" apps and social focus rooms, allowing users to track their personal growth while staying connected with their study group.

### ✨ Key Features

- **Social Connectivity:** A robust friend system allowing users to find "Study Buddies" via email, manage pending requests, and build a dedicated academic circle.
- **Real-Time Presence:** Powered by Supabase, the app features a live "Study Group" sidebar where you can see exactly who is online and focusing in real-time.
- **Gamified Goal Tracking:** A sleek Daily Focus dashboard that categorizes tasks into active and archived states, helping users maintain a clean workspace and a clear mind.
- **Interactive Focus Rooms:** (Based on your code hints) Integrated chat and messaging systems to facilitate peer-to-peer support during deep-work sessions.

### 🛠️ The Tech Stack

- **Frontend:** React 18 with TypeScript for type-safe, scalable component architecture.
- **Styling:** Tailwind CSS for a modern, responsive, and high-performance UI.
- **Backend-as-a-Service:** Supabase for PostgreSQL database management, Auth, and Realtime Presence.
- **Deployment:** Vercel with CI/CD integration for seamless production builds.
